# 蜉蝣哲学：活不过一天也要搅局

一款黑色幽默的 24 小时压缩人生小游戏。玩家扮演一只寿命极短的蜉蝣，在加班、摸鱼、社交、AI 外包和当众搅局之间选择自己的荒诞路线。

## 本地试玩

直接打开 `index.html`，或在当前目录启动静态服务器后访问：

```bash
python3 -m http.server 8766
```

然后打开：

```text
http://127.0.0.1:8766/index.html
```

## 发布说明

这是一个静态网页游戏，核心文件为：

- `index.html`
- `src/game-core.js`

当前版本会从 CDN 加载字体、Tailwind 和 GSAP，因此线上试玩需要联网。
